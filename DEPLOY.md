# Deploy Stakable Token

Full instructions for testing, building, deploying, initializing, and running the staking dApp are in the main **[README.md](./README.md)** — see the section **"Staking DeFi dApp (Svelte + Soroban)"**.

Summary: test (`cargo test -p stakable-token`) → build → deploy → init → mint (reward pool + user) → set `VITE_CONTRACT_ID` in `app/.env` → `npm run dev` in `app`.
